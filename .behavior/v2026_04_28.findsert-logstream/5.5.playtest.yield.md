# playtest: cloudwatch outlet for log collection

## prerequisites

1. **aws credentials configured** — run `rhx keyrack unlock --owner ehmpath --env test`
2. **aws region set** — `AWS_REGION` env var (defaults to `us-east-1`)
3. **repo dependencies** — `npm ci` already run
4. **clean build** — `npm run build` passes

## sandbox

- log groups created in cloudwatch will be named with unique test suffixes
- cleanup: any test log groups are deleted automatically by the test suite
- no os.fileops required for this playtest
